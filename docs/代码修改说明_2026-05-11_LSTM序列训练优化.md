# LSTM 序列训练优化说明

## 修改目标

本次修改针对机器学习模块中 LSTM 效果弱于 Random Forest / LightGBM 的问题，落实以下优化方向：

1. 增加训练可用的历史数据量和 LSTM 序列样本上限。
2. 降低 LSTM 模型复杂度，减少小样本过拟合。
3. 加强 dropout、weight decay 等正则化配置。
4. 将 LSTM 输入窗口改为可配置参数，便于实验 `20 / 30 / 60` 等不同窗口。
5. 给 LSTM 序列输入补充更多市场和行业相关特征。

## 修改文件

- `python/common/config.py`
- `python/ml/stock_ml.py`
- `python/ml/train_daily_predict.py`
- `python/ml/predict_daily.py`

## 具体修改

### 1. 扩大默认训练数据规模

在 `python/common/config.py` 中调整默认机器学习配置：

- `ML_MAX_TRAIN_ROWS` 默认从 `16000` 提高到 `120000`。
- `ML_LSTM_MAX_SEQUENCES` 默认从 `8000` 提高到 `60000`。
- `ML_TRAINING_LOOKBACK_DAYS` 默认从 `730` 提高到 `1500`。

这样在本地数据充足时，LSTM 可以拿到更多连续时间序列样本，而不是过早被默认上限截断。

### 2. 降低 LSTM 模型复杂度

在 `python/ml/stock_ml.py` 中将 LSTM 结构参数改为配置驱动：

- `ML_LSTM_HIDDEN_SIZE` 默认 `64`。
- `ML_LSTM_HEAD_SIZE` 默认 `32`。
- `ML_LSTM_NUM_LAYERS` 默认 `1`。

相比原来的两层 `hidden_size=128`，默认模型更小，更适合当前股票样本规模，降低过拟合风险。

### 3. 加强正则化

新增并使用以下配置：

- `ML_LSTM_DROPOUT`，默认 `0.25`。
- `ML_LSTM_WEIGHT_DECAY`，默认 `0.0005`。
- `ML_LSTM_LEARNING_RATE`，默认 `0.0007`。
- `ML_LSTM_BATCH_SIZE`，默认 `128`。

训练优化器从 `Adam` 改为 `AdamW`，并对价格预测分支和方向分类分支都使用 weight decay。

### 4. 支持可配置 LSTM 窗口

新增配置：

```text
ML_LSTM_WINDOW_SIZE=30
```

`stock_ml.py`、`train_daily_predict.py`、`predict_daily.py` 都改为读取该配置，不再固定写死 `20`。

如需实验不同窗口，可在 PowerShell 中设置：

```powershell
$env:ML_LSTM_WINDOW_SIZE='60'
```

### 5. 增加市场和行业序列特征

在 LSTM 序列特征中新增：

- `change_pct_scaled`
- `sector_avg_change_pct`
- `market_avg_change_pct`
- `sector_momentum_5`
- `market_momentum_5`
- `relative_market_index_return_1`
- `relative_market_index_return_3`

这些特征让 LSTM 不只看单只股票自身走势，也能看到行业平均涨跌、大盘平均涨跌、行业动量以及个股相对市场指数的表现。

## 后续需要重新训练

由于 LSTM 输入特征数量、窗口大小和模型结构都发生了变化，旧的 LSTM 模型文件不再适合作为当前结构的最终结果。需要重新训练模型：

```powershell
D:\anaconda3\envs\MachineLearn\python.exe -m python.ml.train_daily_predict
```

如果想进一步实验更长窗口：

```powershell
$env:ML_LSTM_WINDOW_SIZE='60'
D:\anaconda3\envs\MachineLearn\python.exe -m python.ml.train_daily_predict
```

## 仍需观察的问题

本次修改能改善 LSTM 的训练条件，但不保证 LSTM 一定超过 LightGBM。股票短期方向预测噪声很高，最终仍应以 walk-forward 指标、macro-F1、balanced accuracy 和是否超过多数类基线为准。
