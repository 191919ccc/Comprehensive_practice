# 未使用文件和缓存清理说明

## 修改目的

清理项目中已经不再被运行入口引用的旧前端文件、临时测试脚本和编译缓存，降低后续维护时的误判风险。

## 删除内容

- `frontend/app.js`
  - 早期首页脚本，当前所有 HTML 页面均未引用。
  - 首页运行逻辑已迁移到 `frontend/dashboard-home.js`。

- `frontend/style.css`
  - 早期首页样式，当前所有 HTML 页面均未引用。
  - 首页样式保留在 `frontend/index.html`，子页面样式使用 `frontend/dash-pages.css`。

- `python/tests/111.py`
  - 临时调参和标签分布检查脚本，不是正式单元测试。
  - 正式测试保留 `python/tests/test_stock_utils.py`。

- `python/**/__pycache__/`
  - Python 编译缓存，可由解释器自动重新生成。

- `java-backend/target/recruitment-backend-0.0.1-SNAPSHOT.jar*`
  - 旧名称后端构建产物。
  - 当前 Maven artifact 和启动脚本统一使用 `stock-risk-backend-0.0.1-SNAPSHOT.jar`。

- `java-backend/target/backend_check_stdout.log`
- `java-backend/target/backend_check_stderr.log`
  - 旧检查日志，不参与运行。

## 同步修改

- 更新 `README.md`，移除 `node --check frontend\app.js`。
- 更新 `frontend/README.md`，说明旧 `app.js` 和 `style.css` 已删除。
- 更新项目说明文档中的前端文件清单，改为 `dashboard-home.js`、`shared-ui.js`、`dash-pages.js`、`dash-pages.css`。
