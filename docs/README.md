# 项目文档目录

本目录按文档用途拆分，避免历史修改说明、正式说明文档和答辩材料混在同一层级。

## 目录说明

- `changelog/`：每次代码修改后的说明文档，记录修改目的、涉及文件、验证命令和剩余限制。
- `reports/`：项目说明书、系统说明文档等正式长文档。
- `defense/`：答辩展示、功能介绍、AI 阅读版说明等面向展示的材料。
- `ppt/`：PPT 源文件、预览图和生成清单。

## 维护规则

1. 后续每次代码修改都要在 `docs/changelog/` 新增一份说明。
2. 面向老师或答辩的正式材料放入 `docs/defense/` 或 `docs/reports/`。
3. PPT 文件和 PPT 预览图统一保留在 `docs/ppt/`。
4. 不把模型产物、日志、数据库导出和本机 `.env` 放入文档目录。
