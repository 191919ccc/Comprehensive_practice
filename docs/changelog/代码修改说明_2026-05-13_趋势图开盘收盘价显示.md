# 趋势图开盘价与收盘价显示修改说明

## 修改目标

在首页股票趋势图原有展示基础上，让鼠标悬停到某个时间点时能看到该时间点对应的开盘价和收盘价；日线图同时绘制开盘价折线和收盘价折线，方便对比每天开盘到收盘的变化。

## 修改文件

- `frontend/index.html`
- `java-backend/src/main/java/com/recruitment/backend/service/DashboardService.java`

## 具体修改

1. `frontend/index.html`
   - 增加 `chartTooltipOptions()`，统一给 Chart.js 提示框补充开盘价、收盘价。
   - 实时模式下，鼠标悬停到分钟趋势点时显示该点的 `open_price` 和 `last_price`。
   - 日线模式下，鼠标悬停到某天时显示该天的 `open` 和 `close`。
   - 日线图新增 `日线开盘价` 折线，原有收盘价走势保留为 `日线收盘价`。
   - 图表交互改为按同一个横轴时间点触发，鼠标移动到某天附近即可查看该天对应价格。
   - 移除固定展示的开盘价/收盘价卡片，避免误解为只显示最新一天数据。

2. `DashboardService.java`
   - 在 `/stocks/{symbol}/trend` 分钟趋势接口返回字段中增加 `open_price`。
   - 前端实时模式可以直接从趋势接口读取开盘价，不需要额外请求或改数据库结构。
