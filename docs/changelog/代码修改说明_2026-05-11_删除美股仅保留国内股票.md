# 代码修改说明：删除美股仅保留国内股票

## 修改目标

将实时采集股票池中的美股删除，只保留国内相关市场股票，避免前端下拉框和实时行情展示中混入美股数据。

本次处理保留了 A 股和港股：

- A 股：`SH`、`SZ`
- 港股：`HK`

## 修改文件

| 文件 | 修改内容 |
| --- | --- |
| `python/data/stock_symbols.json` | 删除 `NASDAQ`、`NYSE` 市场股票，仅保留 `SH`、`SZ`、`HK` 股票。 |
| `frontend/index.html` | 股票下拉框增加国内市场过滤，数据库中已有的历史美股记录不会继续混入前端选择列表。 |

## 具体修改

1. 从实时采集配置中删除以下美股类型：
   - 美股科技股
   - 美股新能源股
   - 美股金融股
   - 美股能源股
   - 美股消费股
2. 前端新增 `isDomesticStock()` 判断逻辑：
   - `market` 为 `SH`、`SZ`、`BJ`、`HK` 时保留；
   - 6 位数字代码按 A 股处理；
   - 5 位数字代码按港股处理；
   - 其他如 `AAPL`、`TSLA`、`AMD`、`INTC` 等英文代码不再进入下拉框。

## 注意事项

数据库里以前已经采集过的美股历史记录不会自动删除，但前端下拉框已经过滤，不会再优先展示这些美股。如果后续需要彻底清理数据库中的美股历史行情，可以单独执行 SQL 删除 `NASDAQ`、`NYSE` 市场数据。
