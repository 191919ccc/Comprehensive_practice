# AI 回复格式与链接渲染优化说明

日期：2026-05-21

## 修改目标

解决 AI 回复内容在前端显示为原始 Markdown、排版混乱、新闻链接不可点击的问题。

## 修改文件

- `frontend/index.html`
- `java-backend/src/main/java/com/recruitment/backend/service/AiChatService.java`

## 具体修改

### 1. 前端 Markdown 渲染

原前端虽然引入了 `marked`，但 AI 回复使用 `textContent` 写入页面，因此：

- `###` 会原样显示。
- `**加粗**` 不会生效。
- `[标题](链接)` 不会变成可点击链接。

现在新增：

- `renderMarkdown()`
- `sanitizeHtml()`
- `setAiMessageContent()`

助手回复会通过 `marked.parse()` 渲染成 HTML，并对 HTML 做基础清理：

- 移除 `script/style/iframe/object/embed`。
- 移除 `on*` 事件属性。
- 禁止 `javascript:` 链接。
- 所有链接自动添加 `target="_blank"` 和 `rel="noopener noreferrer"`。

### 2. AI 消息样式优化

新增了 AI 回复区域的标题、列表、链接、加粗样式：

- 标题间距更清楚。
- 列表缩进更稳定。
- 链接使用蓝色并带下划线，可点击打开。
- 助手消息取消 `white-space: pre-wrap`，避免 Markdown 渲染后排版过宽。

### 3. 后端 Prompt 重新约束

`AiChatService` 的 System Prompt 已调整为固定四段结构：

1. `### 核心结论`
2. `### 相关新闻`
3. `### 影响分析`
4. `### 综合建议`

并明确要求：

- 不输出裸 URL。
- 新闻必须使用 `[新闻标题](https://...)` 格式。
- 不允许只罗列链接。
- 必须做基本面、资金面、风险点分析。
- 不输出代码块。
